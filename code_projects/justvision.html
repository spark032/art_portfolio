Just Vision

CSCI 1430 Computer Vision Final Project - Just Dance Recreation

About This Project

Just Vision is a computer vision recreation of the popular video game Just Dance, developed as a final project for CSCI 1430 Computer Vision at Brown University. Using live depth camera input and machine learning techniques, the system captures and analyzes dance movements in real-time to provide interactive gameplay feedback. This project was created by Kamya Raman, Max Smith-Stern, Stephanie Lin, Sue An Park.

Project Description

Our goal was to create an accessible dance game that encourages healthy movement and cognitive activity. The system uses a Microsoft Kinect depth camera to capture user movements and provides real-time visual feedback through an on-screen display. Players can dance along to the YMCA song while the system analyzes their poses and calculates accuracy scores, creating an engaging and entertaining experience for users of all skill levels.

Technical Approach

The project employs a sophisticated three-step computer vision pipeline:

  • Person Isolation: Uses depth thresholding and connected component analysis to separate the dancer from the background
  • Joint Classification: Implements Random Forest classifiers trained on the SURREAL synthetic dataset to predict body part segmentation maps and estimate 24 joint positions using separate regressors for upper and lower body parts
  • Pose Recognition: Applies Random Forest classifiers to the extracted joint vectors to identify and score specific dance poses in real-time

The system processes live depth images, converts them into joint location vectors, and classifies poses with synchronized music playback for an immersive gaming experience.

Broader Applications

The techniques developed for mapping 3D depth data to 2D joint locations extend far beyond entertainment applications. This computer vision approach has significant potential in:

  • Autonomous Vehicle Safety: Predicting pedestrian movement patterns and routes for improved collision avoidance
  • Healthcare & Rehabilitation: Identifying improper movements for injury prevention and monitoring physical therapy progress
  • Sports Analytics: Analyzing athletic performance and technique improvement
  • Accessibility Technology: Creating gesture-based interfaces for individuals with mobility limitations

This project represents the foundation of a broader, more widely integrated technique for human pose recognition and movement analysis in real-world applications.
